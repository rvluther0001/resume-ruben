Apa Aja Yang Gw Kerjain?

BAU & Report Debugging

Tiket BAU (*Business As Usual*) adalah makanan sehari-hari. Gw bertugas nge-debug anomali pada report harian. Ini butuh analisa dalam menyelusuri script PostgreSQL, Python, atau Hive/Impala buat nyari tau kenapa data bisa *mismatch* atau *stuck*, dan tentunya.. langsung di-fix.

Data Pipeline Engineering

Membangun dan me-maintain pipeline ETL/ELT. Gw menggunakan Python dan SQL untuk merubah raw data menjadi format yang bersih dan siap digunakan oleh tim analis atau *business intelligence*.

API Development

Nggak cuma ngerjain data, kadang gw juga ngebangun API sederhana menggunakan GO (Golang). Biasanya ini dibutuhin buat nge-serve data olahan ke *frontend* atau internal service lainnya biar akses datanya lebih *smooth* dan kencang.
